# 小红书笔记模板

## 适用场景
同城获客、本地生活服务、教育招生、电商种草

## 笔记结构

### 标题（必须）
- 20字以内，前5字必须包含核心关键词
- 使用数字增强吸引力（"3个方法"、"7天见效"）
- 制造信息差或好奇心（"原来…"、"终于知道…"、"别再…"）
- 加入地域词增强同城推荐（城市名/区域名）

### 正文结构
```
[开头 Hook — 1-2句]
用痛点或场景代入，让读者觉得"说的就是我"。
避免上来就推销。

[核心内容 — 3-5个要点]
每个要点用 emoji + 加粗标题 + 1-2句解释。
融入真实案例和具体细节（时间、地点、效果数据）。
用第一人称讲故事，不要用说明文语气。

[结尾 CTA — 1-2句]
引导互动（"你们觉得呢？"、"有同样经历的举手🙋"）
或引导私信/关注（不要太硬，用"需要的姐妹可以问我"这种语气）
```

### 标签（5-10个）
- 必含：行业词 + 地域词 + 长尾词
- 格式：#关键词
- 示例：#上海家政 #阿姨推荐 #同城服务 #保洁避坑 #生活小妙招

## 语气要求
- **像朋友在聊天**，不像品牌在发广告
- 可以用口语化表达（"真的绝了"、"姐妹们"、"救命"）
- 适当使用 emoji 增加可读性，但不要过多（每段 1-2 个）
- 避免：营销话术、夸大宣传、绝对化用语（"最好"、"第一"）

## 矩阵变体策略
同一素材生成不同版本时，变化以下维度：
1. **视角切换**：用户视角 → 服务者视角 → 第三方推荐视角
2. **场景切换**：家庭场景 → 职场场景 → 节日场景
3. **表达风格**：干货分享型 → 故事型 → 问答型
4. **标题公式**：数字型 → 反常识型 → 场景痛点型

## 合规红线
- 不使用广告法禁用词（最、第一、绝对等）
- 不进行虚假宣传或承诺效果
- 不使用其他品牌的对比贬低
- 不涉及敏感话题（政治、宗教等）
- 图文需与实际服务一致
