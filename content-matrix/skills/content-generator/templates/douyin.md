# 抖音短视频脚本模板

## 适用场景
同城获客、本地生活服务、教育招生、个人 IP 打造

## 视频结构

### 开头 Hook（前3秒）
- 必须在 3 秒内抓住注意力
- 常用公式：
  - 提问型："你知道为什么……？"
  - 冲突型："千万别再这样做了！"
  - 数字型："3 个方法让你……"
  - 场景型：直接展示服务现场/效果对比

### 正文内容（15-45秒）
```
[痛点共鸣 — 5秒]
说出目标用户的真实困扰，让他们觉得"对对对就是这样"。

[解决方案 — 15-30秒]
用 1-3 个要点展示你的服务/方法/产品。
穿插真实案例和细节。
保持节奏紧凑，不要拖沓。

[结果展示 — 5秒]
用数据或对比展示效果。
```

### 结尾 CTA（最后3秒）
- "关注我，了解更多"
- "评论区告诉我你遇到的问题"
- "点击主页了解详情"

### 文案（发布时配文）
- 简短有力，100字以内
- 包含关键词和话题标签
- 引导互动

### 话题标签（3-5个）
- 必含：#行业词 #地域词
- 可选：#热门话题 #细分长尾词

## 语气要求
- **真实、自然、有感染力**
- 像在跟朋友面对面聊天
- 语速适中，有停顿和节奏感
- 避免念稿感

## 矩阵变体策略
1. **内容角度**：教程型 → 故事型 → 对比测评型
2. **呈现形式**：口播 → 实拍 → 图文混剪
3. **目标人群**：针对不同痛点切入
4. **时间节点**：结合节日/热点/季节

## 合规红线
- 不使用绝对化用语
- 不做效果承诺
- 不贬低同行
- 不涉及敏感话题
- 标明广告合作（如有）
