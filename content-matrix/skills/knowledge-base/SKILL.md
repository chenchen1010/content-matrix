# 知识底座

存放跨项目通用的理论基础和方法论，供其他 skill 在工作流中引用。

## 何时使用

当以下 skill 执行时，应自动读取相关底座文件作为参考：

| 调用方 | 引用场景 | 读取文件 |
|--------|---------|---------|
| **content-generator** | 生成内容时，选择心理杠杆、设计说服结构 | `marketing-psychology-foundation.md` |
| **topic-scout** | 评估选题的说服力和传播潜力 | `marketing-psychology-foundation.md` |

## 当前底座

| 文件 | 内容 | 覆盖范围 |
|------|------|---------|
| `marketing-psychology-foundation.md` | 营销心理学体系 | 15 个域、80+ 理论，含速查矩阵和实战映射 |

## 使用方式

生成内容前，读取对应底座文件，按以下路径使用：

```
1. 明确目标 → 你要用户做什么？（点击/购买/分享/留存/转介绍）
2. 定位阶段 → 用户当前在心理旅程的哪个阶段？
3. 选择杠杆 → 在对应域中挑选 2-3 个最适用的心理机制
4. 设计手法 → 把机制翻译成具体的文案/页面/流程设计
```

## 扩展

未来可添加更多底座文件，如品牌理论、内容结构学、平台算法策略等，统一放在此目录下。
