## 补全转录（在你本机执行）

1. **注册 MCP**（路径见 `content-matrix/skills/tools/search/douyin/README.md`），确保 `mcporter list` 中有 **`douyin_search`** 与 **`douyin`**。
2. **登录与 Cookie**：运行 `douyin-login` / 任务「内容矩阵 · 抖音登录」，保证 `cookies.txt` 有效。
3. **配置 ASR**：在 `content-matrix/.env` 配置火山或阿里云 ASR（`extract_douyin_text` 依赖项）。
4. **搜索（高赞优先）**：
   ```bash
   mcporter call 'douyin_search.search_videos(keyword: "杭州夜校", count: 20, sort_type: 1)'
   mcporter call 'douyin_search.search_videos(keyword: "成人兴趣班", count: 20, sort_type: 1)'
   ```
5. **详情 + 转录**：对选定 `aweme_id` 或分享链执行 `get_video_detail`，再：
   ```bash
   mcporter call 'douyin.extract_douyin_text(share_link: "https://v.douyin.com/xxx/")'
   ```
6. 将返回的口播文本粘贴进 `4-爆款参考/抖音/` 下对应笔记，并把本报告 `status` 改为 `已完成`。
