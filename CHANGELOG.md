# 更新日志

## 0.1.0 — 2026-03-23

首次发布版本。

- 选题侦察（topic-scout）：多平台关键词搜索 + A1-A5 用户分层
- 内容生成（content-generator）：素材 + 选题 → 多平台内容
- 素材管理（material-library）：Obsidian / 飞书双后端
- 平台发布（publish）：小红书、抖音、公众号、视频号
- 微信排版克隆（wechat-layout-clone）：一键提取公众号文章版式
- 主工作流（master-workflow）：选题→素材→生成→发布→回写 全流程编排
- 框架同步机制（framework-sync）：自动检查更新 + 一键同步
- 经验回馈机制（framework-contribute）：引导用户向主框架提交改进
